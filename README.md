# Genk in Numbers

A mobile-first statistics dashboard for **Genk, Belgium** (Limburg). Built with vanilla HTML, CSS, and JavaScript — no build step required.

## Run locally

```bash
# From the project root — any static file server works, for example:
python3 -m http.server 8080
# Then open http://localhost:8080
```

Or open `index.html` directly in a browser.

## Preview on your phone

**Do not** open the jsDelivr/raw GitHub URL directly — Safari will show HTML source code.

Use this link (renders the real page):

**https://htmlpreview.github.io/?https://raw.githubusercontent.com/briskbe/adblue/main/index.html**

For a clean permanent URL, enable [GitHub Pages](https://github.com/briskbe/adblue/settings/pages): Source **Deploy from branch** → **gh-pages** → **/ (root)** → Save. Then open **https://briskbe.github.io/adblue/**

## Data sources

- [Statbel](https://statbel.fgov.be/) — Belgian statistics
- [Genk in cijfers](https://www.genk.be/incijfers) — City dashboards
- [AlleCijfers.be](https://allecijfers.be/gemeente/genk/) — Municipal indicators

Figures cover population, housing, economy, commerce, and culture (2024–2025 where available).
